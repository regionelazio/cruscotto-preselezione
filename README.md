# Dashboard Preselezioni Attive – Regione Lazio

⚠️ **STATO DEL PROGETTO: IN FASE DI TEST**

Applicazione web per la consultazione e l’analisi delle preselezioni attive pubblicate dalla Regione Lazio.

## Finalità

La dashboard consente una lettura sintetica e interattiva delle opportunità di lavoro, facilitando l’accesso alle informazioni attraverso strumenti di filtro, visualizzazione geografica e rappresentazione grafica dei dati.

## Funzionalità principali

- Visualizzazione delle preselezioni attive
- Filtri di ricerca avanzati:
  - Provincia e Comune della sede di lavoro
  - Centro per l’Impiego (CPI)
  - Qualifica professionale
  - Tipologia contrattuale e orario
  - Stato di pubblicazione
  - Scadenza delle candidature
  - **Collocamento mirato (Legge 68/99)**
- Ricerca libera testuale
- Indicatori sintetici (KPI)
- Mappa interattiva territoriale
- Grafici di distribuzione e trend
- Tabella dettagliata delle preselezioni

## Fonte dati

I dati sono acquisiti dinamicamente da un foglio Google Sheets pubblico e aggiornati in tempo reale.

## Caratteristiche tecniche

- Applicazione sviluppata in HTML, CSS e JavaScript (senza framework)
- Elaborazione e filtraggio dati lato client
- Utilizzo di librerie open source:
  - Plotly.js (visualizzazione grafica)
  - Leaflet (mappa interattiva)
  - SheetJS (lettura dati)

## Accesso

La dashboard è disponibile online tramite GitHub Pages.

## Stato del progetto

Il sistema è attualmente in fase di test:
- le funzionalità potrebbero essere soggette a modifiche
- i dati visualizzati potrebbero subire aggiornamenti o variazioni
- non è garantito un utilizzo in contesti produttivi
